# Glossary

---

| Term | Definition |
|------|------------|
| JWT | JSON Web Token — compact, URL-safe token for authentication |
| OAuth 2.0 | Open standard for token-based authorization |
| bcrypt | Password hashing function with salt |
| Rate Limiting | Technique to block excessive requests from same source |
| Refresh Token | Long-lived token to obtain new access tokens |

---
