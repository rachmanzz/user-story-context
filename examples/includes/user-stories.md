# User Stories

---

## US-1: Login with Email & Password

| Field | Value |
|-------|-------|
| **Epic** | EPIC-1 |
| **Priority** | Must |
| **Points** | 3 |
| **Dependencies** | None |

**As a** registered customer  
**I want** to log in using my email and password  
**So that** I can access my account and shop

[AC](acceptance-criteria.md#us-1) | [BDD](bdd-scenarios.md#us-1)

## US-2: Login with Google OAuth

| Field | Value |
|-------|-------|
| **Epic** | EPIC-1 |
| **Priority** | Should |
| **Points** | 5 |
| **Dependencies** | US-1 |

## US-3: Forgot Password

| Field | Value |
|-------|-------|
| **Epic** | EPIC-1 |
| **Priority** | Must |
| **Points** | 3 |
| **Dependencies** | US-1 |

**So that** I can regain access to my account
